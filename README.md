- ```@FrequencyControl``` 接口访问频率控制
- ```@FrequencyControlContainer``` 多限制频控访问
- ```@RedissonLock``` 分布式锁
- ```@TransactionalExecute``` 分布式事务
- ```@Idempotent``` 幂等性检查
